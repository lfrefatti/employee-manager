# Employee Management API

## Requirements

  * pipenv
  * python 3.7

## Activating virtual environment and installing dependencies 

```bash
  >> pipenv install
  >> pipenv shell
```
## Running this service

```bash
  >> python manage.py runserver
```
## Running tests

```bash
  >> python manage.py test
```
